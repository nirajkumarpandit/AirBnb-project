# UrbanStay (Airbnb Style Hotel Listing Platform)

UrbanStay is a full-stack web application inspired by Airbnb where users can browse, create, and manage hotel or property listings. Users can upload images, add reviews, and manage their listings through a secure authentication system.

## Live Demo

[Visit Website](https://urbanstay-ert7.onrender.com)

## Features

* User authentication and authorization
* Create, edit, and delete property listings
* Image upload using Cloudinary
* Review and rating system
* RESTful API architecture
* MVC project structure
* Responsive UI using EJS and Bootstrap

## Installation

Clone the repository

git clone https://github.com/nirajkumarpandit/AirBnb-project.git

Go to project directory

cd AirBnb-project

Install dependencies

npm install

Run the server

node app.js

Open in browser

http://localhost:3000
